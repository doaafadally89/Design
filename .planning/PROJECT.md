# Advisor Marketplace Module

## What This Is

A marketplace module for the advisor dashboard that enables relationship managers and portfolio managers to buy and sell securities (bonds, stocks, mutual funds) for their HNW/UHNW client portfolios. This is an MVP to validate the core trading workflow with interactive HTML interfaces.

## Core Value

Advisors can quickly find, evaluate, and execute trades on securities for their clients' portfolios with confidence in the data they're acting on.

## Requirements

### Validated

(None yet — ship to validate)

### Active

- [ ] Advisors can view market overview with indices and market movers
- [ ] Advisors can browse and search available securities by type (bonds, stocks, mutual funds)
- [ ] Advisors can view detailed information for individual securities
- [ ] Advisors can execute buy/sell trades for client portfolios
- [ ] Advisors can see performance data and key metrics for securities

### Out of Scope

- Rebalancing workflows — deferred to post-MVP
- Real-time market data feeds — MVP uses delayed data
- Mobile application — web-first approach
- Advanced order types — MVP focuses on simple buy/sell

## Context

- Target users: Relationship managers and portfolio managers at a bank
- Client base: High net worth (HNW) and ultra-high net worth (UHNW) individuals
- Reference UI exists showing expected marketplace structure (overview page, listings, detail pages)
- MVP scope focused on validating core trading workflow
- Tech stack: Interactive HTML interfaces

## Constraints

- **Tech stack**: HTML interactive — client-side interactivity preferred
- **Timeline**: MVP iteration — validate before expanding scope
- **Data**: MVP can use simulated or delayed market data
- **Scope**: Three security types only (bonds, stocks, mutual funds) for initial release
